# steam-tool

This is PowerShell script that finds your Steam installation and tries to downgrade it to an older
(32-bit) version by downloading and applying some files.
Basically: if new updates are causing issues, this might help 👍

How to use
Step1:open your steam 
Step2:Right Click on the folder (Run with powershell)
Step3:Wait to finishing the download
Step4:restart your steam
Enjoy now you have steam(32bit) 

How it works
Checks if TEMP folder exists (creates it if needed)
Looks for Steam installation
Downloads required files
Applies the changes

Enjoy :)
